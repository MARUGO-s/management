Recipe Box 設定画面の使い方
=====================================

■ 設定画面の開き方
1. トップ画面の右上にある「設定」ボタンをクリック
2. 設定モーダルが表示されます

■ 設定項目の説明
=====================================

【AI API設定】
-----------
使用するAI APIを選択します。

・Groq（推奨）
  - 高速なレシピ解析
  - 画像からレシピを抽出する際に使用
  - デフォルトで選択されています

・ChatGPT
  - 自然な日本語での解析
  - より高精度な解析が可能

【AI創作アシスタント設定】
------------------------
AI創作機能で使用するAPIを選択します。

・ChatGPT（推奨）
  - 高品質なレシピ創作
  - 新しいレシピを生成する際に使用
  - デフォルトで選択されています

・Groq
  - 高速なレシピ創作
  - より高速な処理が可能

【Groqモデル設定（URLインポート・翻訳用）】
------------------------------------
URLインポートや翻訳機能で使用するGroqモデルを選択します。

・Gemini 1.5 Flash（推奨）
  - Google Generative Language API
  - バランスの取れた性能
  - デフォルトで選択されています

・Llama 3.1 8B Instant
  - Groq高速モデル
  - 14.4K回/日、500Kトークン/日

・Llama 3.3 70B Versatile
  - 高精度モデル
  - 1K回/日、100Kトークン/日

・Gemma2 9B IT
  - Google製モデル
  - 14.4K回/日、500Kトークン/日

・GPT-OSS 120B
  - 超高精度モデル
  - 1K回/日、200Kトークン/日

・GPT-OSS 20B
  - 高精度モデル
  - 1K回/日、200Kトークン/日

■ 設定の保存方法
=====================================

1. 設定を変更した後、「保存」ボタンをクリック
2. 「保存完了」メッセージが表示されます
3. 1.5秒後に自動的に設定画面が閉じます
4. 設定は自動的に保存されます

■ 設定画面の閉じ方
=====================================

以下の方法で設定画面を閉じることができます：

1. 「保存」ボタンをクリック（設定が保存されます）
2. 「キャンセル」ボタンをクリック（変更は保存されません）
3. 右上の「×」ボタンをクリック（変更は保存されません）
4. モーダル外の背景をクリック（変更は保存されません）

■ 推奨設定
=====================================

【一般的な使用】
・AI API: Groq
・AI創作API: ChatGPT
・Groqモデル: Gemini 1.5 Flash

【高速処理重視】
・AI API: Groq
・AI創作API: Groq
・Groqモデル: Llama 3.1 8B Instant

【高精度重視】
・AI API: ChatGPT
・AI創作API: ChatGPT
・Groqモデル: Llama 3.3 70B Versatile

■ 使用制限について
=====================================

各API・モデルごとに1日あたりの使用回数とトークン数に制限があります。

・Groqモデル: 1K〜14.4K回/日、100K〜500Kトークン/日
・ChatGPT API: プランによって異なります

制限に達した場合は、別のモデルに切り替えるか、翌日まで待つ必要があります。

■ トラブルシューティング
=====================================

【設定画面が開かない場合】
1. ブラウザの開発者ツール（F12）を開く
2. Consoleタブでエラーメッセージを確認
3. ページを再読み込みして再試行

【設定が保存されない場合】
1. ブラウザのローカルストレージが有効か確認
2. プライベートブラウジングモードでないか確認
3. ブラウザのキャッシュをクリア

【文字が見切れる場合】
1. ブラウザのズームレベルを100%に設定
2. ブラウザのウィンドウサイズを調整
3. ページを再読み込み

■ 注意事項
=====================================

・設定はブラウザのローカルストレージに保存されます
・異なるブラウザやデバイスでは設定が共有されません
・設定を変更した後は必ず「保存」ボタンをクリックしてください
・設定の変更は即座に反映されます

■ サポート
=====================================

設定に関する問題が発生した場合は、以下を確認してください：

1. ブラウザの開発者ツール（F12）のConsoleタブでエラーメッセージを確認
2. ページを再読み込みして再試行
3. ブラウザのキャッシュをクリア
4. 別のブラウザで試行
